Detailed Description

Script that handles Square object spawning and Setting Camera. Changes number of squares spawned based off of chosen difficulty and Camera perspective based off of chosen handedness.

Function run when a collision is detected with collider at bottom of screen. If a triangle collides with the collider at the bottom of the screen, the game is ended and the game over screen is loaded.
